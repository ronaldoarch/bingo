==========================================
VERIFICAR ONDE ESTÃO OS ARQUIVOS
==========================================

Execute no servidor para verificar:

# 1. Verificar estrutura completa
cd ~/public_html
ls -la
pwd

# 2. Verificar se há subdiretórios
find . -name "index.html" -type f

# 3. Verificar estrutura de diretórios
tree -L 3
# OU se não tiver tree:
ls -laR | grep -E "^d|index.html"

# 4. Verificar se os arquivos estão diretamente em public_html
ls -la ~/public_html/index.html
ls -la ~/public_html/static/

==========================================
SE OS ARQUIVOS ESTÃO EM SUBDIRETÓRIO:
==========================================

# Se encontrar index.html em subdiretório, mover arquivos:

cd ~/public_html
